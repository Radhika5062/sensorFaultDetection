1. Login to AWS Console
2. Create IAM user for deployment with specifig access
    a) EC2 access: It is virtual machine. 
    b) S3 bucket: To store artifact and model in S3 bucket
    c) ECR: Elastic Container Registry to save your docker image in AWS

    Description about deployment
    a) Build docker image of the source code
    b) Push your docker image to ECR
    c) Launch EC2
    d) Pull your image from ECR into EC2
    e) Launch your docker image in EC2

    Policy:
    a) AmazonEC2ContainerRegistryFullAccess
    b) AmazonEC2FullAccess
    c)AmazonS3FullAccess
3. Create a S3 bucket in your region
    bucket name: scania-sensor-pipeline #whatever you want to use.
4. ECR repo to store/save docker image

5. EC2 machine ubuntu created
6. Open EC2 and install docker in EC2 machine
    #optional
    sudo apt-get update -y
    sudo apt-get upgrade

    #required
    curl -fsSL https://get.docker.com -o get docker.sh
    sudo sh get-docker.sh
    sudo usermod -aG docker ubuntu
    newgrp docker
